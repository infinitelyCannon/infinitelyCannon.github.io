Catformer

Catformer is a 2D infinite jumping platformer. In this project we aimed for a challenging experience that is simple to execute. Which gave us more space to focus on stylistic improvements, like diverse cat options and visually appealing graphics, as additional pull-ins.

My roles for this game included:

  • • Gameplay system for random generation of platforms
  • • Repeatedly scrolling parallax effect with background sprites
  • • Implementing Primary jumping mechanic
  • • Oversaw Android platform compilation

The experience of working on this game introduced me to several challengs when it comes to making a game look and perform well on a range of mobile plaforms.

4715 Jetsam

A third person shooter for the Windows platform. The aim for this game was to provide a short, cohesive experience with simple approachable mechanics. Working on this project was a significant lesson in values of team contributions and consice documentation. As well as how much the Unity engine does and does't supply for you when branching from a common set of player mechanics.

My roles for this game included:

  • • Implementing player movement and combat mechanics
  • • Designing a finite state machine for enemy AI
  • • Creating functionality for a console terminal-based user menu
